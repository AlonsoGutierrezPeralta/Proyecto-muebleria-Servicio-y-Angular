# Sistema de Mueblería – API REST + Angular

Proyecto académico que implementa un sistema de gestión de muebles mediante una arquitectura cliente-servidor.

## Descripción
El proyecto consiste en:
- Un backend desarrollado en Java con Spring Boot que expone servicios REST.
- Un frontend desarrollado en Angular que consume dichos servicios.
- Persistencia de datos en una base de datos relacional.

## Arquitectura
- Backend: Spring Boot (API REST)
- Frontend: Angular
- Base de datos: MySQL

## Funcionalidades
- Gestión de muebles (crear, listar, actualizar y eliminar).
- Validaciones básicas de datos.
- Comunicación frontend-backend mediante API REST.

## Estructura del repositorio
- `MobiliariaSpring-1/` - Backend (Spring Boot)
- `MuebleriaAngular/` - Frontend (Angular)
- `mobiliaria.sql` - Script de base de datos

## Tecnologías utilizadas
- Java
- Spring Boot
- Angular
- MySQL
- REST API

## Contexto
Proyecto desarrollado con fines académicos para aplicar conceptos de desarrollo backend, frontend e integración de sistemas.
